## Interpreter-tier cross-engine micro-bench

Subprocess wall-clock; times in **ms**, **median of N=10 timed
runs** after 1 discarded warmup run. `*` flags a fixture
whose max-min spread exceeded 10% — treat that cell as noisy.

All JIT engines run JIT-disabled (interpreter tier only): v8
`--jitless`, sm `--no-baseline --no-ion`, jsc `JSC_useJIT=0`.
Internal regression compass — not published.

Host: `Darwin 25.5.0 arm64`. Generated by `tools/bench-cross.sh`.

Snapshot at `74c2d0a` (post the lazy property bag Phase 3 +
shape-aware gates [`0cab149`, `6d96854`], the JSObject.get
synth-accessor read for hardened-mode iterators [`8e311c3`],
upsert graduation to default-on [`f48b7b0`], and the N=5→N=10
sample-budget bump [`b0f11bd`]).

Δ vs the prior `4e0c80c` snapshot (the sample-budget bump from
N=5→N=10 contributes a uniform noise-floor reduction; the
fixture-by-fixture shifts below are real-signal on top of that):

- **`object_alloc` 59 → 47 ms (−20 %)** — lazy property bag
  Phase 3 (`0cab149`) skips the per-property `properties.put`
  on shape-mode writes. **Cynic now leads QuickJS-NG on this
  fixture: 47 vs 54 ms** (was 59 vs 56 ms — tied / fractionally
  behind). Within the doc's 15-25 % target band.
- **`class_instantiate` 131 → 123 ms (−6 %)** — same Phase 3
  effect. The IC fast path already wrote slots directly, so
  the slow-path savings are smaller; the lazy-bag doc's
  forecast for this fixture was the wider gap closer
  (~131 → ~80 ms via a deeper lazy-bag pass), still ahead.
  Now ~2× behind QJS (123 vs 63), down from 2.1×.
- **`prop_access` 11 → 12 ms (+9 %)** — small regression
  from routing reads through the shape-aware
  `JSObject.lookupOwn` path instead of the simple
  `properties.get` bag read. Still leads QJS (12 vs 15) and
  matches Hermes (12).
- **`array_iter` 22 → 21 ms (−5 %)** — drift in noise band;
  no shape-attributable cause.
- Other fixtures within noise of the prior snapshot.

Cynic-vs-QuickJS-NG headline: **leads or matches on 8 of 11
fixtures** (arith_loop, array_iter, json_stringify,
**object_alloc** (newly ahead), prop_access, prop_write,
string_concat — plus tail_recursion which QJS can't run).
Remaining gaps: `class_instantiate` (cynic 123 vs qjs 63 —
narrowed to ~2×), `method_call` (33 vs 18 — same gap as
prior snapshot), `promise_chain` (13 vs 11 — within noise).

**`tail_recursion`** runs only on cynic and JSC — the two
engines shipping ES2015 §15.10 Proper Tail Calls. Cynic
88 ms vs JSC 20 ms; JSC's 4.4× lead reflects their JIT tiers
operating on the tail-call-flagged opcode, while cynic stays
in interpreter dispatch. Closing the gap is a JIT-tier task.
